# Route42 Branding Assets

- `route42-launcher-premium-1024.png` - final launcher master used to generate Android app icons.
- `route42-playstore-512.png` - ready-to-use 512x512 Play Store icon export.

The original imported source image is not stored in the public repository. The branding source of truth in this repo is `route42-launcher-premium-1024.png`.

Android resources are generated from `route42-launcher-premium-1024.png` and stored in:

- `app/src/main/res/drawable/route42icon.png`
- `app/src/main/res/mipmap-*/ic_launcher.png`
- `app/src/main/res/mipmap-*/ic_launcher_round.png`
- `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`
- `app/src/main/res/mipmap-anydpi-v26/ic_launcher_round.xml`
