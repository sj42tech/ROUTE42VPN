## Positioning Notes

Route42 is an Android client for self-hosted or provider-issued profiles. It is not a public VPN service and should not be positioned as a generic free VPN app.

Good query targets:

- `android vless client`
- `self hosted vpn android`
- `sing-box android client`
- `import vless link android`
- `android vpn client with custom routing`
